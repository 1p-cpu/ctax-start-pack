02  C-Tax Start Packとは

最初から、
経営に使える会計データをつくる。

C-Tax Start Packは、個人事業主・小規模法人・法人成り前後の事業者を対象とした、税務顧問の入口商品です。

freeeとAIを活用して会計・税務の業務を標準化することで、導入しやすい価格設計を実現しています。その分、納税予測・資金繰り・経営判断の支援に時間を使います。

自己流の仕訳を後から直すより、最初からルールを整える方が、長い目で見てはるかに効率的です。freeeの設定・仕訳ルール・月次の確認フローを最初に設計することで、税務だけでなく、利益・納税・資金繰りが見える経営の土台をつくります。

申告書を作るだけの関与ではありません。数字を経営の武器にするための、最初の一歩です。

03  C-Taxが大切にしている考え方

「AIは思考を支援し、人は未来を動かす」

freeeとAIを活用した業務の標準化は、手段に過ぎません。その余力を使って私たちが集中するのは、経営者の意思決定を支えることです。

税務顧問を「過去の申告作業の代行」ではなく、「将来の法人化・融資・事業拡大に向けた経営基盤づくり」として位置づけています。会計データを経営の武器にするための設計から始める。それがC-Taxのスタンスです。

04  会計ソフトについて

freee 第一推奨。
マネーフォワード クラウドにも対応可能。

C-Tax Start Packは、freeeを第一推奨としています。クラウド上でリアルタイムに帳簿を確認でき、取引の自動取り込みや仕訳候補の活用、ルール設計との親和性が高く、私たちが標準化している業務フローと最も相性がよいためです。freeeをお使いでない方には、初期設定から丁寧にサポートします。

マネーフォワード クラウドについて
すでにご利用中の場合や、事業内容・経理体制との相性上こちらが適している場合にも対応可能です。ただし、C-Tax Start Packの標準フローはfreeeを前提として設計されているため、業務量・設定状況・確認範囲に応じて個別にご相談ください。

06  AI活用型記帳サポート

ルールを育てながら、
帳簿を経営に使える状態へ。

自己流での入力を後から直すより、最初からルールを整えた方が、経営に使えるデータが育ちます。C-Tax Start Packでは、AI活用型記帳サポートとして、取引データの取り込みから仕訳ルールの設計・改善までをサポートします。

対応内容

  • 領収書・請求書・通帳明細・クレジットカード明細などをfreeeに取り込む
  • 取引内容に応じた仕訳候補・登録ルールを設計・確認する
  • 定型取引はルール化し、入力の手間を継続的に減らしていく
  • 不明点は一覧化してお客様に確認する（都度の丸投げではなく、ルールとして整理する）
  • 月次レビュー前に帳簿を確認できる状態へ整える
  • 将来的な自動化を見据えた会計運用を設計する

料金：取引量・資料の整理状況・確認範囲に応じて、月額料金に加算または個別お見積りとなります。

別途または対応外となるもの

  • 紙領収書の大量整理・スキャン代行
  • 過年度分の一括整理
  • 現金出納帳の復元
  • 私用・事業用混在の大量整理
  • 証憑不足の状態での帳簿復元
  • インボイス不備の個別確認大量対応

08  対象となる方

全員のための
サービスでは、ありません。

✓  こんな方に向いています
  • ✓個人事業主・フリーランスとして事業を継続している
  • ✓freeeを使い始めたが、正しく運用できているか不安
  • ✓自己流の帳簿から抜け出して、最初から正しい会計の土台を整えたい
  • ✓法人成りしたばかりで、会計・税務の基盤を整えたい
  • ✓確定申告だけでなく、納税・資金繰り・利益を早めに把握したい
  • ✓将来的に融資・採用・事業拡大を視野に入れている
  • ✓経営の数字を見ながら事業を伸ばしていきたい
—  向いていない方
  • —費用を最小限に抑えることを最優先にされている
  • —自己流の仕訳入力はそのままに、数字だけ出してほしい
  • —紙の領収書や通帳をすべて渡せばあとはやってもらえると考えている
  • —クラウド会計を使うご意向がない
  • —節税対策のみを目的にされている
  • —経営の数字を定期的に確認することに関心がない
